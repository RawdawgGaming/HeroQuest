# Hero Quest – Game Design Document

## Overview

Hero Quest is a side-scrolling action RPG inspired by Castle Crashers. Players select a hero class and progress through stages filled with enemies, minibosses, bosses, mounts, companions, loot, and evolving abilities.

Each stage begins with a short cutscene showing the world boss taunting the player before escaping.

---

## Core Gameplay Loop

### Primary Loop

Enter Stage  
Fight enemy waves  
Collect gold + XP + drops  
Defeat miniboss  
Reach final boss  
Earn rewards  
Upgrade hero / gear / mount / sidekick  
Enter next stage
